# AgenticChain Finance

Antigravity Finance MVP using Streamlit. A multi-page application that simulates a trust loop between a Manufacturer, an AI Auditor, and a Lender. 

## Features
- Auditor Agent using LangChain to ingest and reason about simulated manufacturing data.
- Blockchain Ledgersimulation with SHA-256 hashes of verified production milestones.
- Dynamic Live Credit Score reflecting the AI Agent's findings.
- Industrial Fintech UI/UX.

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
